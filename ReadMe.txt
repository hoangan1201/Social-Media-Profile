Final project COM 212 Spring 2022
This is a console-based program that simulates a Social Media Profile. It allows creating a new profile or loading an existing profile, and keeps track of that profile by storing data in a local .txt file. Users can add or delete friends, add events, and add posts to timeline.
To run program:
1. Compile MySocialProfile.java file
---> javac MySocialProfile.java
2. Run Main class
---> java Main
